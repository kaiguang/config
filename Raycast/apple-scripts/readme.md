Add this directory to the Raycast Script Commands.

Check the `windowName` in each script, you will need to manually change the Google Chrome window name to match it. Tips: currently this is set up in the macOS keyboard shortcut for Google Chrome, ctrl-n to 'Name Window...'.

Map the Script Command to the desired shortcut you want, for me it's hyper-g for my personal profile and hyper-t for work profile.
